# CRM SDR / Desafio Attus

Documentação geral e referências do desafio estão na pasta `docs/`. A aplicação Angular do desafio prático fica em **`desafio-attus/`**.

## Aplicação (listagem de usuários)

1. `cd desafio-attus`
2. `npm install`
3. `npm start`
4. Acesse `http://localhost:4200/` (rota principal: `/usuarios`).

Detalhes, testes, lint e notas de ambiente: [desafio-attus/README.md](desafio-attus/README.md).

PDF do desafio: [desafio_frontend_attus (1).pdf](desafio_frontend_attus%20(1).pdf).
